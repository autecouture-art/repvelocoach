# iPhone/モバイル最適化ガイド

## 実装内容

### 1. PWA対応（Progressive Web App）
- **ホーム画面に追加可能**: Safariの「ホーム画面に追加」でアプリのように使用可能
- **オフライン対応**: Service Workerでキャッシュ機能
- **フルスクリーン表示**: ブラウザUIを非表示にしてアプリのような体験

### 2. モバイルUI最適化
- **大きなタッチターゲット**: ボタンは最小44px（iOS推奨サイズ）
- **フォントサイズ最適化**: 16px以上で自動ズームを防止
- **レスポンシブデザイン**: 画面サイズに応じて自動調整
- **タッチ操作最適化**: ダブルタップズーム無効化

### 3. iPhone向け最適化
- **Safe Area対応**: ノッチやホームインジケーターを考慮
- **ステータスバー**: 黒半透明で統一感のある表示
- **画面ロック防止**: 自動ロックを防ぐ設定推奨

## 使用方法

### ホーム画面に追加

1. Safariで http://localhost:5001 にアクセス
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面にアイコンが追加されます

### ジムでの使用推奨設定

1. **画面ロック防止**
   - 設定 > 画面表示と明るさ > 自動ロック > なし
   - または、トレーニング中は画面を常時点灯

2. **Wi-Fi接続**
   - サーバーと同じネットワークに接続
   - または、サーバーのIPアドレス（例: 192.168.0.172:5001）に直接アクセス

3. **バッテリー節約**
   - 低電力モードをオフ
   - 画面の明るさを適度に調整

## ネットワーク設定

### ローカルネットワークでのアクセス

サーバー起動時に表示されるIPアドレスを使用：
```
http://192.168.0.172:5001
```

### セキュリティ注意事項

- ローカルネットワーク内でのみアクセス可能
- 本番環境ではHTTPSと認証を実装推奨

## パフォーマンス最適化

- **Service Worker**: 静的リソースをキャッシュ
- **軽量なライブラリ**: CDNから読み込み
- **効率的なデータ送信**: WebSocketでリアルタイム通信

## トラブルシューティング

### BLE接続ができない
- サーバーとiPhoneが同じネットワークに接続されているか確認
- サーバーのログを確認

### 画面が小さく見える
- Safariの表示モードを確認
- ホーム画面から起動するとフルスクリーン表示

### オフラインで動作しない
- Service Workerが正しく登録されているか確認
- キャッシュがクリアされていないか確認
